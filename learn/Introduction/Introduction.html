Hello! Welcome to first lesson of JavaScript. let's talk about JavaScript.

The first thing I wanted to say is that JavaScript ≠ Java. Many confuse these two languages and they don't see the difference between them. This is what Java looks like:

            
    class HelloWorld{
            public static void  main (String args []) {
            System.out.println("Hello World");
            } 
        }

JavaScript looks like:

    
    <script>
        alert("Hello World!");
    </script>

Secondly: JavaScript - implementation of the ECMAScript standard.

Now we pass to where we will write Javascript. Our development environment will be the place where we wrote HTML. I work in Brackets. Let's code!

Where We must write our code? In HTML. Set tag <script> before </body>. Like here:


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>JS Learn</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
         <p>JS - cool!</p>
    <script></script>
    </body>
</html>

You can write js in HTML or in file. But you must write this tag like that:


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>JS Learn</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
         <p>JS - cool!</p>
    <script src="name_of_JSscript.js"></script>
    </body>
</html>

In order for us to connect modern JS (on which we will work) we need to write at the beginning of each script "use strict";

And then the script will connect from the file. Lets make your first program on JS. The first thing JS can do is change the HTML code. How do we do this? Very easy. We must use .innerHTML. For example:


<script>
    "use strict";
    elem.innerHTML = "someText";
</script>

In this code we change inner HTML code of element elem. How element elem now looks like this:


<p id="elem">someText</p>

elem - it is id of HTML tag. In JS we always use id, we can't use classes! There is ful code(HTML + JS):


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>JS Learn</title>
    </head>
    <body>
        <p id="elem">JS - cool!</p>
    <script>
        "use strict";
        elem.innerHTML = "someText";
    </script>
    </body>
</html>

Ok, we see in our page someText. But we don't see previous text. We can not only replace, but also add elements. We can write elem.innerHTML = elem.innerHTML + "someText"; or elem.innerHTML += "someText";((a = a + b) == (a += b))


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>JS Learn</title>
    </head>
    <body>
        <p id="elem">JS - cool!</p>
    <script>
        "use strict";
        elem.innerHTML += " someText";
    </script>
    </body>
</html>

result of First example(elem.innerHTML = "someText"): someText

result of Second example(elem.innerHTML += " someText";): JS-cool! someText

Let's add interactive. Now we must add button by tag <button></button>. Let the id of this button be btn. And text will be - click. So, our button is <button id="btn">click</button>. Let's add our button to our page.


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>JS Learn</title>
    </head>
    <body>
        <button id="btn">click</p>
        <p id="elem">JS - cool!</p>
    <script>
        "use strict";
        elem.innerHTML += " someText";
    </script>
    </body>
</html>

Now we need script for our button. We can add the event listener on the button. btn.addEventListener("click", function(){//some_code}); - add event listener on our button.For example: I want to make a clicker. I need one variable, button and text. To declare a variable, use - let. All variables in JS are dynamically typed. Let's write simple clicker


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>JS Learn</title>
    </head>
    <body>
        <button id="btn">click</button>
        <p id="elem">JS - cool!</p>
        
    <script>
        "use strict";
        let clickCount = 0;//Declare a variable
        btn.addEventListener("click", function(){//Add Click listener
            clickCount++;//clickCount += 1
            elem.innerHTML = clickCount;//update information
        });
    </script>
    
    </body>
</html>

Also you can add event listener to other tags(div, p, etc.). for example: divId.addEventListener("click", function(){});

Congrats! Now you create a web-clicker! You can style your game, and make her more interesting!
